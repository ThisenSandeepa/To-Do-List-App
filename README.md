# To-Do List App

## Description

This is a simple and user-friendly To-Do List web application built using HTML, CSS, and JavaScript. The app allows users to manage their daily tasks efficiently. Users can add new tasks, edit or delete existing tasks, and mark tasks as completed. The app also includes filter options to view all tasks, only pending tasks, or only completed tasks. All data is stored in the browser's local storage, so tasks remain saved even after closing or refreshing the page.

## Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as complete/incomplete
- Filter tasks (All, Pending, Completed)
- Clear all completed tasks at once
- Data persistence using localStorage
- Keyboard support (Press Enter to add task)

## Technologies Used

- HTML
- CSS
- JavaScript
